Customer Behaviour Analysis Dashboard
Project Overview

This project focuses on analyzing customer demographics, spending behavior, and customer segmentation using Power BI.

The objective is to help businesses better understand customer groups, identify high-value customers, and support data-driven decision-making.

Business Problem

Businesses need to understand:

Who are their most valuable customers?
Which customer groups contribute the most revenue?
How do demographics influence spending behavior?
How can customer segments be targeted more effectively?

This dashboard provides insights into customer behavior patterns through interactive visualizations.

Key Findings
Regular customers represent the majority of the customer base.
Married customers generate the highest overall spending.
Graduation-level customers contribute the highest revenue.
VIP customers represent a smaller segment but contribute significantly to business value.
